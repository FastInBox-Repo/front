# FastInBox Front

Frontend inicial da plataforma FastInBox, configurado com Next.js + TypeScript.

## Scripts

```bash
npm run dev
npm run build
npm run lint
```

## Ambiente local

- Frontend: `http://localhost:3001`
- Backend esperado: `http://localhost:4001`
- Proxy local: `/backend/*`
